STARTUP EXPLORER

Startup Explorer is a great way to find out which startups
are where. Find the startups near you - or plan your
startup trek in another city!


With Startup Explorer you can search the major metro areas, filter on funding and company size, and view lists or maps to find all the awesome companies in the area. The best feature is that all the data is curated and maintained by real people at Buddy.com. This is not driven through machine learning or some fancy voodoo, it's good old fashioned hard work which gives us the best startup list available.


All of the data is available through the Buddy.com BaaS API solution for free.
